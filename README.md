# 科系猜猜樂遊玩指南
***
## **遊玩前**
1. 請先下載departments.js以及GAME.html
2. 確認兩個檔案放在同一資料夾
3. 確認完畢後即可開始遊戲啦!!!

## **開始遊戲**
1. 心中默想一個科系
2. 點選開始遊戲
3. 依據問題點選「是」、「否」或「我不知道」
4. 查詢結果
